This domain is established to be used for illustrative examples in documents or application identifiers. You may use this domain in examples without prior coordination or asking for permission.

Typical Use Cases

  • Use as the domain part of Bundle Identifier (iOS) / Application ID (Android) for open source projects. Such as dev.yourcompany.MyApp.
  • Use as a placeholder value of the App's TextField. Such as JohnDoe@yourcompany.dev.
  • Use as a dummy account of the Screenshot, Document and Test Cases.

About This Domain

This domain is managed by an individual. Although the RFC defines example.* domains for such purposes, I think yourcompany.dev is much cooler than example.* domains. :)
